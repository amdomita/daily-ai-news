# 每日AI与科技资讯 - 2026-04-25

- **Meta 推出 Muse Spark** — 首席AI官Alexandr Wang主导的Superintelligence Labs发布首个旗舰大模型，在多模态感知、推理、医疗健康和智能体任务上达到竞争级别，训练计算成本远低于Llama 4同尺寸变体。（CNBC）

- **OpenClaw 开源项目爆红** — 年初至今从9000 Star暴涨至超21万 Star，2026年GitHub增速最快项目，专注隐私优先的自托管AI推理，满足企业不愿数据上云的需求。（GitHub / ByteByteGo Blog）

## 数据库与中间件

- **Apache Kafka 4.2.0 已正式发布**（2026年2月）— 全面完成KRaft架构迁移，彻底移除ZooKeeper依赖；前版4.1已引入Queues预览（KIP-932）、新Streams重平衡协议（KIP-1071）、插件自定义指标注册（KIP-877）等特性。（Apache Kafka Blog / Confluent Blog）

- **Redis Enterprise for Kubernetes 7.8.6-14 发布**（2026年4月）— Kubernetes环境企业版专项更新；Redis 8主线新增Vector Set、时序、Bloom Filter等8类数据结构，部分命令性能提升87%，已将Redis Stack与社区版合并为统一发行包。（Redis Docs）

- **MySQL 9.3 Innovation Release 持续迭代** — 新增Vector数据类型、JavaScript存储过程（企业版）、EXPLAIN ANALYZE JSON输出等特性；生产稳定场景仍推荐MySQL 8.4 LTS；Percona宣布不跟进9.x Innovation版本。（MySQL官方文档 / Percona Blog）

## 云原生

- **Kubernetes v1.36 "Haru" 正式发布**（4月22日）— 共70项增强，18项升至GA。亮点：HPA Scale-to-Zero默认启用、MutatingAdmissionPolicy GA、User Namespaces稳定、OCI VolumeSource GA；移除存在root权限风险的gitRepo卷；废弃Ingress NGINX（已于3月24日停止维护）。（Kubernetes官方博客）

- **CNCF：Kubernetes 不足以独立保障 LLM 工作负载安全** — CNCF指出K8s原有安全机制无法防御提示注入、数据意外暴露、工具滥用等AI特有威胁，呼吁在K8s上部署LLM时须额外构建专属安全层。（InfoQ / CNCF Blog）

- **KubeCon Europe 2026 聚焦 AI 与数字主权** — 各厂商围绕Kubernetes抽象层、GPU管理与自动化展开，核心议题是推动AI工作负载从PoC走向生产，欧洲数字主权问题引发广泛讨论。（Cloud Native Now）

- **Kubernetes v1.36 细粒度 kubelet API 授权 GA** — SIG Auth与SIG Node联合宣布fine-grained kubelet API authorization正式GA，显著提升节点级访问控制粒度，缩小横向移动攻击面。（Kubernetes官方博客）
