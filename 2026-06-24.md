# 每日AI与科技资讯 - 2026-06-24

## AI/人工智能

- **智谱 GLM-5.2 以 MIT 协议开源，744B 参数 MoE 模型位居全球第二** — 智谱 AI 发布 GLM-5.2，拥有 744B 参数（40B 激活）、100万 Token 无损上下文窗口，Code Arena 排名开源第一、全球第二，是国内头部 AI 实验室迄今最宽松的开源授权。（知乎）

- **MiniMax M3 开源发布，SWE-Bench Pro 开源模型第一** — MiniMax 发布开源权重模型 M3，在 SWE-Bench Pro 以 59.0% 领跑开源阵营，融合原生多模态计算机操控能力与 100万 Token 上下文，基于 MiniMax Sparse Attention 架构。（devFlokers）

- **Claude Fable 5 登顶全球多项权威榜单** — Anthropic Claude Fable 5（6月9日发布）将上下文扩展至 200万 Token，在 AA Intelligence Index（65分）和 LMArena 文本生成排行榜（Elo 1510）均列第一，SWE-bench Pro 得分 80.3%。（CSDN AI Agent社区）

- **6月第一周超 25 款开源模型集中发布，覆盖文本、图像、音视频与 3D** — 分析人士认为这是"开源 vs 闭源"的历史性拐点，本地部署高性能多模态模型对个人开发者已触手可及。（AI工具宝箱）

- **里约热内卢市政 IT 公司发布 Rio 3.5 Open 397B，超越 DeepSeek V4 Pro** — IplanRIO 在 Hugging Face 以 MIT 协议发布 Rio 3.5 Open 397B，在 Terminal-Bench 2.1 上超越 DeepSeek V4 Pro，意味着顶级开源模型不再是大型 AI 实验室的专属。（devFlokers）

- **OpenCode v1.17.8 发布** — 热门开源 AI 编程工具 OpenCode 持续迭代，新版优化会话时间线加载速度并改进 OpenAI 兼容 Provider 支持。（GitHub Releases）

## 数据库与中间件

- **MySQL 9.7.0 LTS 发布，首次将企业级特性开放给社区版** — Oracle 于 4月21日发布 MySQL 9.7.0 LTS（6月出 9.7.1 补丁），新增数据库内 JavaScript 执行、JSON Duality View DML、动态数据脱敏、OpenID Connect 认证及新一代 Hypergraph 查询优化器，支持周期至 2034 年。（Oracle MySQL Blog / InfoQ）

- **Redis 披露 3 个 RCE 高危漏洞，需立即升级补丁版本** — CVE-2026-23479（阻塞客户端 use-after-free）、CVE-2026-23631 "DarkReplica"（复制子系统 use-after-free）等 5 个 CVE 影响 Redis 7.2.x-8.6.x，已修复版本（7.2.14、7.4.9、8.2.6、8.4.3、8.6.3）于 5月5日发布，自托管用户须立即升级。（SecurityOnline / GBHackers / CISA）

- **Apache Kafka 4.3.0 发布，含 25 个 KIP 和 600+ 提交** — 亮点包括 Broker 日志目录封锁（Cordoning）、分层存储 Follower-Fetch 优化及新 Coordinator 缓冲区配置；Streams Scala 模块已标记废弃，计划在 Kafka 5.0 移除。（Apache Kafka Blog / Confluent）

- **Apache RocketMQ 5.5.0 正式可用** — 继 2025 年 11 月发布 5.4.0 后，RocketMQ 5.5.0 现已上线官方下载页，持续专注云原生消息与流处理场景。（RocketMQ 官网 / GitHub）

- **CISA 周报（SB26-159）将 Redis 列为中间件高危攻击面** — CISA 6月1日周安全公告将 Redis 等中间件产品列为重点修复目标，再次强调数据库与中间件基础设施仍是 2026 年中段的主要攻击入口。（CISA）
