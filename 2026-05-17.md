## 云原生

- **Kubernetes v1.36 "Haru" 发布，安全与 AI 工作负载双重强化** — 共70项增强：User Namespaces（容器隔离）、Mutating Admission Policies（CEL 声明式变更）、细粒度 Kubelet API 授权全部晋升 Stable；HPA 支持缩容至零副本；DRA 动态资源分配扩展至内存和 CPU。（Kubernetes.io / InfoQ）

- **Ingress NGINX 正式退役** — Kubernetes SIG Network 于3月24日宣布 Ingress NGINX 项目退役，集群运维团队需迁移至 Gateway API 或其他 Ingress 实现。（Kubernetes.io）

- **Docker 推出 Kanvas 平台，挑战 Helm 与 Kustomize** — 新平台可将 Docker Compose 文件自动转换为 Kubernetes 资源清单，打通本地开发与云端生产环境，降低 K8s 部署门槛。（InfoQ）

- **2026 年 Kubernetes AI 工作负载趋势：GPU 优先与 FinOps 融合** — GPU 成为默认算力范式，DRA 绑定条件晋升 Beta 以支持 AI/ML 高需求场景；AIOps 驱动的 FinOps 技术成为容器资源优化标配。（Fairwinds Blog / Komodor Blog）
