## 数据库与中间件

- **Redis Software 8.0.18-36 安全更新（2026年5月）** — 修复三项高危漏洞：CVE-2026-23479（客户端释放后重用，可致 RCE）、CVE-2026-25243（RESTORE 命令非法内存访问）、CVE-2026-23631（Lua 释放后重用），建议尽快升级（Redis Docs）

- **Apache Kafka 4.3.0 发布** — 完全去除 ZooKeeper 依赖，全面运行于 KRaft 模式，Share Groups 进入 GA，Kafka Streams 服务端再平衡协议正式可用（Apache Kafka Blog）

- **Redis 8.8 新增原生数组类型** — 引入 Native Array Type 与新命令集，带来显著性能提升（pbxscience.com）

- **MySQL 8.4.9 维护版本发布** — 4月7日发布，为 8.4 LTS 系列最新版本，持续强化安全默认配置与生产稳定性（VersionLog）
