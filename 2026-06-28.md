# 每日AI与科技资讯 - 2026-06-28

## AI/人工智能
- **Claude Fable 5 & Mythos 5 发布** — Anthropic 推出两款新旗舰模型，上下文窗口突破 200 万 token，多项基准测试全球第一；Claude Fable 5 仅用 1 天完成了需耗时两个月的 5000 万行代码迁移任务（Anthropic）
- **MiniMax M3 开源权重模型发布** — 428B 参数 MoE 架构，支持 100 万 token 上下文、原生多模态及桌面计算机操控，SWE-Bench Pro 得分 59.0%，首个同时具备前沿编程能力、超长上下文与原生多模态的完全开放模型（MiniMax / The Decoder）
- **NVIDIA Nemotron 3 Ultra 开源发布** — 550B 参数混合 Mamba-MoE 架构，支持 100 万 token 上下文，以 OpenMDW 1.1 协议同时开放权重、训练数据与训练方案，推理吞吐量比 Kimi K2.6 高 4.8 倍（NVIDIA Newsroom）
- **OpenAI × Broadcom 联合发布 "Jalapeño" AI 推理芯片** — 首款自研 LLM 推理加速芯片从设计到流片仅用 9 个月，成本较现有 GPU 方案降低约 50%，预计 2026 年底开始部署（TechCrunch / CNBC）
- **Kimi K2.7 Code 开源编程模型发布** — 月之暗面发布 1T 参数 MoE 模型，256K 上下文，编程基准较上代提升最高 31.5%，推理 token 消耗减少 30%（Moonshot AI）

## 开源软件
- **Headroom — AI Agent 上下文压缩库登顶 GitHub 趋势榜** — 一周内新增 14,266 颗星，可将 tool 输出、日志、RAG 块等压缩 60–95% 的 token 且不损失答案质量，支持库、代理或 MCP 服务器三种部署方式（GitHub）
- **GLM-5.2 以 MIT 协议完全开源** — 智谱 AI 以最宽松的 MIT 协议开源 GLM-5.2，加入六月开源模型大爆发浪潮，同期超过 25 个涵盖 LLM、图像、音频、视频、3D 生成的模型集中发布（AI Tool Lab）
- **Ideogram 4 以 Apache 2.0 协议开源** — 图像生成模型 Ideogram 4 开放 9.3B 参数 DiT（扩散 Transformer）权重，采用商业友好的 Apache 2.0 协议（AI Tool Lab）
- **NVIDIA Nemotron 3 Ultra 成迄今最强完全开源大模型** — 区别于众多"开放权重"模型，同时开放权重、训练数据与完整训练方案，被视为美国实验室发布的能力最强完全开源模型（Artificial Analysis）
- **Microsoft AutoGen 社区分叉为 AG2 项目** — AutoGen v0.4 的破坏性重写引发社区反弹，原核心开发者以 AG2 名称独立维护，成为 2026 年开源治理典型案例（devFlokers）

- **First Commerce Bank 迁移至 FIS Horizon 平台** — 美国新泽西州银行迁移核心系统，重点引入标准化数据接口和代理式 AI 商业工具，加速 AI 能力落地（FinTech Futures）
- **Interswitch Group 采用 Temenos 套件向非洲扩张云托管服务** — 覆盖核心银行、支付、财富管理、数字银行和金融犯罪防控，向非洲各地银行机构提供完整托管服务（FinTech Futures）

## 数据库与中间件
- **MySQL 8.4.10 & 9.7.1 LTS 发布** — 8.4.10 为关键安全补丁更新，9.7.1 为最新 LTS 版本，支持周期延至 2034 年 4 月（MySQL Release Notes）
- **Redis 8.8.0 发布，修复两处 RCE 漏洞** — 修复"unblock client 流程中的 Use-After-Free"和"RESTORE 命令中的非法内存访问"两个可导致远程代码执行的严重漏洞，同期 8.2.7、8.4.4、8.6.4 维护版本一并发布（Redis）
- **Redis Insight 3.6.0 GA 发布** — 全面支持 Redis 8 原生向量数据类型 Vector Sets，提供完整 UI 管理界面（Redis Insight GitHub）
- **Apache Kafka 4.3.1 发布** — 修复约 15 个问题，重点解决 Kafka Streams RocksDB 原生内存泄漏关键 Bug（Apache Kafka）
- **Apache RocketMQ 5.5.0 引入 AI Agent 会话管理能力** — 新增百万级 LiteTopics，专为 AI Agent 会话状态管理设计，支持 MCP 协议与长会话状态续传（Apache RocketMQ GitHub）

## 云原生
- **Kubernetes 1.36.2 / 1.35.6 / 1.34.9 补丁发布** — 6 月 9 日发布三个维护版本补丁，同日 1.33 版本宣告终止生命周期（kubernetes.io）
- **Docker Engine 29.6.0 发布，修复两项安全漏洞** — 修复 CVE-2026-8936（grpcfuse 导致 VM panic）和 CVE-2026-5817（Model Runner vllm-metal 后端容器逃逸 RCE），新增 Model Runner 镜像仓库镜像支持（Docker Docs）
- **containerd 集中披露六项高危漏洞** — 涉及 CDI 注解注入、宿主机 root 命令执行、本地镜像 tag 污染、symlink 跟随导致宿主机任意文件读取等，建议立即升级（containerd GitHub Security Advisories）
- **GKE 为 1.30–1.36 全系列推送安全补丁** — 6 月 19–20 日对 Container-Optimized OS 及 Ubuntu 节点镜像同步推送安全修复（Google Cloud）
- **OpenTelemetry 正式毕业成为 CNCF 顶级项目** — 6 月 17 日 CNCF 宣布毕业，确认其作为云原生可观测性事实标准地位（CNCF）
- **KubeCon India 2026 召开，印度云原生开发者达 225 万** — CNCF 与 SlashData 联合报告确认印度已成全球最大云原生社区之一，大会同日宣布新增 14 家白银会员（PRNewswire）
