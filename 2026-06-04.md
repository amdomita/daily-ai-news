# 每日AI与科技资讯 - 2026-06-04

## AI/人工智能

- **Microsoft Build 2026：MAI-Thinking-1 首款自研推理模型正式亮相** — 微软在 Build 2026 上发布 MAI-Thinking-1，拥有 350 亿活跃参数、256K 上下文窗口，AIME 2026 得分 94.5%，SWE-Bench Pro 成绩比肩 Claude Opus 4.6，完全不依赖 OpenAI 数据；同步发布 50 亿参数编程模型 MAI-Code-1-Flash，已于 6 月 2 日向 GitHub Copilot 用户推送（CNBC / Microsoft AI）

- **Anthropic 秘密提交 IPO 申请，估值逼近万亿美元** — 6 月 1 日，Claude 母公司 Anthropic 向 SEC 提交保密 IPO 申请，年化收入达 470 亿美元，最新估值约 9,650 亿美元，预计成为今年三大万亿美元 IPO 之一（CNN / CNBC）

- **自主 AI 工具发现 Redis 隐藏两年的 RCE 漏洞** — 自主安全扫描 AI 工具主动发现潜伏 Redis 代码库两年以上的 CVE-2026-23479（Use-After-Free 远程代码执行），展现 AI 在漏洞挖掘领域的实战能力（The Hacker News）

## 数据库与中间件

- **Redis 五项高危 CVE 安全补丁发布，建议立即升级** — 官方修复两个 Use-After-Free（CVE-2026-23479、CVE-2026-23631）及三个 RESTORE 命令内存越界漏洞（CVE-2026-25243/25588/25589），CVSS 最高 7.7，可被认证用户触发 RCE；Redis Cloud 已自动升级，自托管用户须手动升级（redis.io）

- **Apache Kafka 4.3.0：KRaft 架构成熟，彻底告别 ZooKeeper** — 4.3.0 进一步完善元数据仲裁机制，社区正式进入 KRaft-only 维护轨道；4.2.1 同步修复关键 Bug（Apache Kafka）

- **Apache Iceberg 1.11.0 内置表加密，数据湖安全能力补齐关键缺口** — 通过信封加密与 Google KMS 集成，无需第三方工具即可实现静态数据表级加密，适配 Spark 4.1 与 Flink 2.1（InfoQ）
