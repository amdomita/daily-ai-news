# 每日AI与科技资讯 - 2026-04-29

## AI/人工智能

- **OpenAI 发布 GPT-5.5** — 继 GPT-5.4 不到两个月后推出，编码、计算机操作和深度研究能力大幅提升（CNBC）
- **DeepSeek 发布 V4 预览版** — 1.6万亿参数 MoE 模型（Pro 版激活参数49B），在编码基准测试中声称顶级性能，推理与 Agent 任务能力显著提升（CNBC / Bloomberg）
- **Google 开源 Gemma 4（Apache 2.0）** — 旗舰版 31B Dense 模型在 Arena AI 开源模型榜单位居第三，是迄今最强开源权重发布之一（llm-stats.com）
- **阿里 Qwen 3 发布，覆盖8种参数规模** — 支持灵活部署场景，被评为当前最优开源模型选项之一（Fazm Blog）
- **GLM-5.1 以 MIT 协议发布** — 754B 参数 MoE 模型，专为长周期 Agent 任务设计，4月7日发布（Fazm Blog）
- **Anthropic 确认 Claude Mythos 存在但不公开** — 被称为 Anthropic 有史以来最强模型，4月7日确认，不对外发布（renovateqr.com）

- **OpenClaw 突破 30 万 GitHub Star** — 自托管 AI 平台创下 GitHub 历史上最快增长记录之一，与 n8n（18万+ Star）共同引领自托管 AI 复兴浪潮（Fazm Blog）
- **BlackMagic AI 开源 GTM 工程平台（4月28日）** — 正式开源其 GTM 自动化平台，定位为 Clay 的开源替代方案（OpenPR）
- **Nous Research Hermes Agent v0.8.0 发布** — 合并209个 PR、关闭82个 issue，持续位居 GitHub 趋势榜首（devFlokers）

## 银行金融科技

- **Customers Bank 与 OpenAI 达成战略合作，打造 AI 原生银行** — 合作涵盖贷款、存款、支付等商业银行核心业务，其支付平台年处理规模达2万亿美元（Fintech Global）
- **Agentic AI 正在重塑银行合规体系** — 分析显示77%的银行已软启动生成式AI，但仅31%部署了 Agentic AI；自主决策系统正在深度渗透合规流程（Fintech Global）
- **英国财政部特别委员会发布 AI 金融服务监管回应** — 建议在2026年底前将主要 AI 和云服务商列为关键第三方（CTP），强化金融科技依赖监管（Regulation Tomorrow）
- **美国 GENIUS 法案确立联邦稳定币监管框架** — 国会通过该法案，为支付稳定币建立联邦级综合监管体系，是金融科技监管历史性转折（Cleary Gottlieb）

- **Helm v4.1.4 安全补丁发布** — 同期公告 v4.1.5 与 v3.20.3 将于4月8日发布，下一个 Minor 版本 v4.2.0 计划于5月13日推出（helm.sh / GitHub）
- **CNCF 推出 `.project` 标准化元数据规范** — 面向所有 CNCF 托管项目统一项目健康度、治理及生命周期信息的机器可读格式（CNCF contributors blog）
