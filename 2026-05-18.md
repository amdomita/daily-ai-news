# 每日AI与科技资讯 - 2026-05-18

## AI/人工智能

- **GPT-5.5 Instant 成为 ChatGPT 新默认模型** — OpenAI 于5月5日将 GPT-5.5 Instant 设为 ChatGPT 默认模型，同步发布三款实时语音模型：GPT-Realtime-2（支持128K上下文）、GPT-Realtime-Translate（覆盖70+语言实时翻译）、GPT-Realtime-Whisper（实时转录字幕），语音交互进入新纪元。（163.com / WhatLLM.org）

- **SubQ 1M-Preview 首个商用次二次方注意力架构 LLM 发布** — Subquadratic 公司推出 SubQ，采用全新稀疏注意力架构替代标准 Transformer，原生支持1200万 Token 超长上下文，推理成本约为主流前沿模型的1/5，被视为架构层面的重要突破。（WhatLLM.org）

- **Kimi K2.6 开源登顶 SWE-Bench Pro** — 月之暗面4月下旬正式开源 Kimi K2.6，在软件工程权威测试 SWE-Bench Pro 中以58.6分拿下全球第一，国产开源模型首次登顶软件工程领域世界排行榜。（知乎 / 机器之心）

- **Hugging Face 报告：中国模型占全球下载量41%** — Hugging Face 2026年春季开源AI生态报告显示，该平台上近四成下载量来自中国研发模型，中国开源大模型生态全球影响力持续扩大。（llm-stats.com）

- **Gemini 3.1 Flash Lite 发布** — Google 于5月8日推出轻量级版本 Gemini 3.1 Flash Lite，面向低延迟、低成本应用场景，进一步完善 Gemini 3.1 产品矩阵。（WhatLLM.org）

- **xAI Grok 4.3 扩大公测** — xAI 将 Grok 4.3 向更大范围用户开放，多项基准测试表现持续提升，加速追赶 GPT 与 Gemini 系列。（Air Street Press）

## 数据库与中间件

- **Redis 发布三版本修复多个高危 RCE 漏洞** — Redis 于5月5日同步发布 8.2.6、8.4.3、8.6.3 版本，修复多个严重安全漏洞，包括 CVE-2026-23479（释放后使用可致远程代码执行）、CVE-2026-25243（RESTORE 指令内存越界 RCE）及 CVE-2026-23631（Lua 释放后使用 RCE），建议用户立即升级至 8.6.3。（redis.io / GitHub）

- **Apache Kafka 4.2.0 为当前最新稳定版** — Kafka 4.2.0 于2月17日发布，目前维护4个活跃支持版本（4.2/4.1/4.0/3.9），项目保持每年3个版本节奏，KRaft 模式持续成熟。（Apache Kafka / endoflife.date）

- **Spring for Apache Kafka 发布首个候选版本** — 2026年4月第三周，Spring for Apache Kafka 随 Spring Boot、Spring Security 等组成的 Spring 生态系列发布首个候选版本，为主要版本升级做准备。（InfoQ）

- **Grafana 13 引入 Loki Kafka 后端架构** — Grafana Labs 在 GrafanaCON 2026 巴塞罗那大会上发布 Grafana 13，Loki 日志系统新增 Kafka 作为摄取层后端，大幅提升高吞吐日志场景下的可扩展性。（Kai Waehner Blog）

- **DigitalOcean 托管数据库支持 Prometheus 指标抓取** — DigitalOcean 宣布旗下 PostgreSQL、MySQL、Redis、Kafka 托管数据库服务全面支持可抓取 Prometheus 指标，便于统一监控接入。（DigitalOcean Blog）

## 云原生

- **Helm v4 正式发布，六年来首次重大版本升级** — CNCF 在 Helm 项目10周年之际发布 Helm 4，重点改进测试自动化、Chart 签名、性能及分发机制，是 Kubernetes 包管理器六年来首个 Major 版本。（DEVOPSdigest / CNCF）

- **Kyverno 1.18 发布，CNCF 毕业后首个版本** — 政策即代码引擎 Kyverno 发布1.18版本，这是其3月正式从 CNCF 毕业后的首个 Release，增强安全性、扩展 CLI 能力，助力团队实现策略驱动的大规模 Kubernetes 运维。（CNCF Blog）

- **CNCF Kubernetes AI 合规认证平台数量翻近一番** — CNCF Kubernetes AI 合规认证项目自去年11月启动以来，认证平台从18个增至31个，计划推出自动化合规测试机器人进行第三方验证，推动 AI 工作负载标准化落地。（PR Newswire）

- **KubeCon + CloudNativeCon India 2026 确认6月举办** — KubeCon India 2026 定于6月18-19日在孟买举行，这是 CNCF 首次将旗舰会议带入印度，标志着南亚云原生社区的快速崛起。（CNCF）

- **CNCF 调查：Kubernetes 为 AI 基础设施核心支柱** — CNCF 最新报告确认 Kubernetes 已成为 AI 基础设施的基础技术，在受访组织中使用率领先，Helm（81%）、Prometheus（77%）、containerd（74%）紧随其后。（The New Stack / CNCF）
