## 数据库与中间件

- **Redis 8.6.2 发布** — 最新稳定版于 2026 年 3 月 24 日发布，持续优化性能与稳定性（VersionLog / endoflife.date）

- **Kafka 4.3.0 代码冻结，计划 5 月发布** — 代码冻结节点为 4 月 8 日，此前 4.0.2（修复 43 个问题）与 4.1.2（修复 31 个问题）均于 3 月中旬发布，含多项阻塞性 Bug 修复（Red Hat Developer）

- **MySQL 9.3 发布，查询优化器大幅增强** — Innovation 版本新增对 `>ANY / >=ANY / <ALL` 等全部比较操作符的子查询转派生表（subquery_to_derived）优化支持，SELECT 和 WHERE 子句均受益；同时引入 VECTOR 数据类型持续完善（MySQL Docs / Bytebase）

- **RocketMQ 5.5.0 持续迭代** — 最新稳定版 5.5.0 已发布，2026 年继续推进消息队列选择器改进、分层存储修复及命令验证增强，客户端 Java SDK 于 1 月完成年内首次发版（apache/rocketmq GitHub）

- **KubeBlocks 支持统一管理 MySQL/Redis/RocketMQ 等多种数据库** — 开源 Kubernetes Operator 项目 KubeBlocks 持续活跃，覆盖主流关系型、缓存、消息队列数据库的云原生生命周期管理（apecloud/kubeblocks GitHub）

## 云原生

- **KubeCon + CloudNativeCon Europe 2026 正式开幕** — 本届大会主题聚焦"AI 时代的 Kubernetes"，Kubernetes 正从容器编排平台演进为企业 AI 部署、治理与规模化运营的控制平面（CNCF）

- **Istio 为 AI 时代重大演进：Ambient 多集群 Beta + 推理网关** — CNCF 宣布 Istio 进入 AI 时代新阶段，新增 Ambient 模式多集群 Beta 支持、Gateway API Inference Extension，以及实验性 agentgateway 集成，专为 LLM 推理流量设计（InfoQ）

- **Broadcom 将 Velero 捐献至 CNCF Sandbox** — Kubernetes 备份与灾难恢复工具 Velero 正式进入 CNCF 孵化阶段，降低企业云原生基础设施运维成本（CNCF）

- **AWS KubeCon 主题演讲：让 Kubernetes 变得"隐形"** — AWS 以"从复杂到清晰"为题，展示简化 Kubernetes 采用路径的最新努力，当前最新稳定版为 Kubernetes v1.35.4 / Docker Engine v29.2.1（The New Stack）
